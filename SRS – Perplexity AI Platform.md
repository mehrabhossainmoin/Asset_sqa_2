## 📖 Project Overview
This platform replicates and enhances the core functionalities of Perplexity AI, delivering real-time, source-backed answers to user queries by synthesizing web search data with large language models (LLMs).

## ✨ Key Features
- **Conversational Search:** Context-aware, follow-up query handling.
- **Source Citation:** Automatic linking to web references for transparency.
- **Multi-Modal Support:** Image/File uploads for contextual queries (Planned).
- **Focus Modes:** Narrow down searches to Academic, Writing, YouTube, or Reddit.

## 💻 Tech Stack
- **Frontend:** Next.js / React.js, Tailwind CSS
- **Backend:** Node.js / FastAPI
- **Database:** PostgreSQL / MongoDB
- **AI/LLM:** Google Gemini API (Gemini 1.5 Pro/Flash) / LangChain

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/mehrabhossainmoin/Asset_sqa_2.git
   ```
2. Install NPM packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
